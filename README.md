                                                                                  
  Что изменить в проекте вар1 для вар2                                                                                                                                                                        
  1. Модели — виды транспорта                                                                                                                                                                                 
  App/models.py — в Application.TRANSPORT_CHOICES:
  ('boat', 'Катер')          →  ('bus', 'Автобус')
  ('cruise', 'Круизный лайнер') →  ('electric_bus', 'Электробус')
  ('yacht', 'Яхта')          →  ('tram', 'Трамвай')

  2. Шаблоны — название портала

  Во всех файлах заменить Водить.РФ на Пассажирам.РФ:
  - App/templates/App/base.html — название в шапке и подвале, <title>
  - App/templates/App/index.html — заголовок, описание
  - App/templates/App/register.html — title
  - App/templates/App/login.html — title
  - App/templates/App/application_form.html — подпись
  - App/templates/App/profile.html — title

  3. Шаблоны — карточки преимуществ

  index.html — заменить блок из 3 карточек:
  🛥️  Катера        →  🚌 Автобусы
  🚢 Круизные лайнеры →  🔌 Электробусы
  ⛵ Яхты           →  🚊 Трамваи
  Тексты описаний под каждым — под городские пассажирские перевозки.

  4. Шаблоны — alt-тексты слайдера

  index.html и profile.html — заменить подписи изображений:
  Катер / Круизный лайнер / Яхта / Речной транспорт
  → Автобус / Электробус / Трамвай / Пассажирский транспорт

  5. Статика — изображения слайдера

  4 изображения в static/images/ должны быть связаны с городским пассажирским транспортом, а не с
  водным. Либо переименовать/заменить файлы slide-1.jpg...slide-4.jpg.

  6. Руководство по стилю

  В вар2 должно быть своё руководство (если есть). Нужно сверить цвета — если другие, обновить
  tailwind.config в base.html (цвета sea-blue, turquoise, forest-green, metal-gray).

  ---
  Итог: менять модели, логику, вьюхи, админку, формы — не нужно. Всё идентично вар1. Только названия,
  виды транспорта и изображения.
